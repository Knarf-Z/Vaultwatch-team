Project in preparation

Do DeFi Bug Bounties Scale with Value at Risk?

This project studies whether public DeFi bug bounty ceilings are calibrated to the amount of value that protocols publicly report as being at risk. We compare Immunefi-listed bounty programs with public protocol metrics such as TVL, protocol category, chain, critical reward cap, reward formula, proof-of-concept requirements, KYC requirements, and scope restrictions.

The project is an empirical public-evidence audit. It is not a claim that any specific protocol is vulnerable, and it is not a private security review. The goal is to measure whether public bounty policies appear proportional to public exposure, and to identify design patterns that make responsible disclosure more or less credible.

  • Data sources: public bug bounty pages, DeFiLlama protocol data, project documentation, and governance records.
  • Main metric: maximum critical bounty divided by a public value-at-risk proxy.
  • Output: cleaned dataset, reproducible figures, and a research note or working paper.
